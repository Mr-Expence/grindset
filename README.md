# Grindset

## A simple website to create a small home workout plan. Can be made into a web-app

### Install as an App
#### iPhone / iPad
- Open GrindSet in Safari
- Tap the Share button
- Tap Add to Home Screen
- Press Add

The app will now appear on your home screen like a native app.

#### Android
- Open GrindSet in Chrome
- Tap the ⋮ menu
- Tap Install App or Add to Home Screen
- Press Install

The app will now appear on your home screen.
